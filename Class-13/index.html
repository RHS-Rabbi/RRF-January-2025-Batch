JavaScript Data Types

Data Type Description Example
String Represents textual data enclosed in quotes. "Hello" or 'World'
Number Represents numeric values (both integers and floating-point). 42, 3.14, -5
Boolean Represents logical values, either true or false. true, false
Undefined A variable that has been declared but not assigned a value. let x; (x is undefined)
Null Represents an explicitly empty or non-existent value. let x = null;
Object A complex data structure used to store collections of data. {name: "John", age: 30}
Array A special type of object for ordered collections. [1, 2, 3] or ["a", "b", "c"]
Function Represents executable code. function() { return 5; }
Symbol Represents a unique and immutable identifier. Symbol("id")
BigInt Represents integers beyond the safe limit for Number. BigInt(9007199254740991n)

JavaScript Variable Declaration Methods

Declaration Method Description Example
var Used to declare variables. The scope is function-level or global. It is hoisted but not block-scoped. var x = 10;
let Used to declare variables. The scope is block-level, and it cannot be redeclared within the same scope. let y = 20;
const Used to declare constants. The value cannot be reassigned, and the scope is block-level. const z = 30;
