Js Data Type And Variable

Data Types

Data Type Category Desctiption Example
Number Primitive Represents both integer and floating-point numbers 42, 3.14
BigInt Primitive Represents integers larger than the Number type can safely store. 123n
String Primitive Represents textual data. Strings are immutable. "Hello"
Boolean Primitive Represents true or false. true, false
Undefined Primitive A variable that has been declared but not assigned a value. undefined
Null Primitive Represents an intentional absence of any value. null
Symbol Primitive Introduced in ES6, Symbol represents a unique and immutable value. Symbol("id")
Object Non-Primitive Represents a collection of key-value pairs. {name: "John"}
Array Non-Primitive A special type of object used for storing ordered collections of data. [1, 2, 3]
Function Non-Primitive Functions are objects and can be treated as values. function() {}
Date Non-Primitive Represents dates and times.
Date Non-Primitive Represents dates and times. new Date()
Regular Expressions Non-Primitive Represents patterns for text matching. /abc/
Map Non-Primitive A collection of key-value pairs where keys can be of any type. new Map()
Set Non-Primitive A collection of unique values. new Set()

variable

Variable Desctiption Example
var This is the oldest way to declare variables. It has function scope, which means it is only visible within the function it is declared in. var x = 10;
let Introduced in ES6 (ES2015), let allows you to declare variables with block scope, meaning the variable is only visible within the block it is declared in. let y = 20;
const Also introduced in ES6, const is used to declare variables that cannot be reassigned after their initial assignment. It also has block scope. const z = 30;
